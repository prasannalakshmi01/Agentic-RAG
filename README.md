# Agentic RAG System using LangGraph

## Overview
This project implements an Agentic RAG pipeline with decision-making, self-correction, and evaluation using LangGraph.

## Features
- LLM-based query routing
- Document relevance grading (LLM-as-judge)
- Query rewriting
- Hallucination detection
- Retry mechanism

## Tech Stack
LangGraph, LangChain, FAISS, Gemini, Streamlit
